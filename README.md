# Sistema-de-gest-o-de-estoque-Pedro_henrique_nascimento_lins
Um sistema simples de gestão de estoque utilizando o Python e SQlite. 
o Sistema tem as seguintes funções: Cadastrar itens, atualizar estoque(retirar ou adicionar), vizualizar estoque e sair do programa.
Ele usa um simples sistema de banco de dados chamdo SQlite3, ele permite que o sistema salve os itens cadastrado mesmo após finalizar o programa. Ele é recomendado para sistemas mais simples, e é mais fácil de usar que outras ferramentas de banco de dados.
